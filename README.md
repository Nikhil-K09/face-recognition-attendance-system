# Installation
Use Python 3.9 version to avoid face_recognition issues

Create virtual environment
```python
python -m venv .venv
```
```python
.venv\Scripts\activate
```
```python
pip install -r requirements.txt
```
# Run
To register new face
```python
py add_face.py
```
Update in database
```python
py encoding.py
```
Live-Recognition using webcam
```python
py recognition.py
```
Webpage preview
```python
py app.py
```
Use GUI
```python
py main.py
```
